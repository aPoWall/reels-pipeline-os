pipeline – как монтировать reels

reels pipeline os держит render-grade spine: source audit, transcript, word-safe cuts, EDL, audio master, overlays, review MP4, text-free base и QA. Любая реализация может заменить renderer, но сохраняет один переносимый editing state.

input
raw `.mov/.mp4`, optional screen proof, creative brief, reference mechanics, publish goal.
analysis
ffprobe source audit, orientation, camera metadata, audio channels, frame scan, source map, transcript with word timings and pause map.
editorial
source blocks, hook/setup/turn/proof/closer roles, one audio spine, word-safe cut points, baseline before experiments.
state
`edit_decision_list.json` as SSOT: `cuts[]`, `overlay_tracks[]`, `render.outputs`, branch status and QA decisions.
surface
caption, card, rail, stamp, proof and cover lanes. copy is viewer-facing, safe-zone aware and public-safe.
outputs
review MP4, final 1080x1920, text-free base, contact sheet, QA report, public recipe without private media.

EDL layer

editing state как SSOT

shaper v2 · 2026-06-15

рабочая последовательность

  1. step 0creative brief / mode selection

    определить цель, нарративную арку, cut policy, overlay density and branch matrix.

  2. step 1deep source analysis

    ffprobe, ориентация, камера, аудио-каналы, frame scan, source map, аудио-источник и visual-only blocks.

  3. step 2transcript + word-level timeline

    слова, паузы, punch words, concept map. cut points ставятся в паузах, не внутри слов.

  4. step 3clean baseline

    silence trim, filler/repeat cleanup, handles 100-260ms, closer tail. baseline сохраняется перед стилем.

  5. step 4branch matrix

    минимум: dry_preserve, caption_only, context branch, optional social_short. Для 10+ вариантов сначала SPECS/render_queue, финалы последовательно.

  6. step 5overlay lanes

    caption, card, rail, stamp, proof. Viewer-facing copy, no internal labels, no source filenames, no `render/QA/branch` на экране.

  7. step 6render + QA

    review MP4, text-free base, contact sheet, ffprobe, volumedetect, first 3 seconds, safe-zone and no-overlap checks.

режимы и где они применяются

modeработаdefault outputважный риск
dry_preserveисходник уже живойtrim only, no overlayне убить физические beat-ы
caption_onlyсоциальный defaultнижние sparse captionsне закрыть лицо / объект
proof_overlayнужен screen / terminal / site proofодин нижний proof blockне превратить в production label
visible_flowпроцесс важнее одной фразыtimeline rail / graphне перегрузить интерфейсом

типовые branch recipes

color return
монохромный или тихий старт → возвращение цвета → restrained captions → clear payoff.
object proof
один физический объект или экранный фрагмент держит доказательство, captions редкие и viewer-facing.
dual voice
coach / critic / meta lanes, no-overlap rules, persona pill в safe-zone, short cuts усиливают voice switch.
collab interruption
interruptions and handoffs как сюжетные beat-ы. turn_map и collabBeats[] обязательны.
cover set
10 cover options: identity references + motif object + clean no-text bases + Pillow caption layer.

libraries stack – из чего собрать pipeline

стек делится на четыре слоя: core cli для проверки и рендера, python libraries для субтитров и кадров, timeline adapters для переносимого state, editor surface для визуального контроля.

core cli

ffmpeg / ffprobe

source audit, crop, scale, concat, audio extraction, AAC, bt709 tags, final diagnostics.

iphone ingest

Apple avconvert

macOS-native path for iPhone HDR/HLG to SDR bt709 before creative grading and render.

speech timing

Whisper / Scribe

word timings, pause map, subtitle base, retranscribe QA after final render.

python image

Pillow / numpy

PNG captions, word highlight frames, contact sheets, masks, thumbnails and QA stills.

subtitle lanes

ASS / SRT / WebVTT

portable caption layer for social default, review text and optional burn-in renders.

state model

EDL JSON + OpenTimelineIO

small public-safe EDL now, OTIO adapter later for richer NLE interchange and metadata.

react render

Remotion

React compositions from typed tracks/items, Player preview, inputProps and deterministic render path.

html render

HyperFrames

plain HTML/CSS/media with timing data, seekable animations, CLI validation and FFmpeg output.

browser preview

WebCodecs / OpenCut

local decode/encode and editor/headless split for future visual reel dashboard.

automation

MCP / CLI tools

structured preflight, transcript, EDL build, render, verify and publish handoff.

quality gates

volumedetect / silencedetect

audio health, dead-air hints, first-frame checks, seam listening and final ffprobe report.

package docs

Markdown + JSON Schema

STACK.md, CONTRIBUTING, branch README, EDL skeletons and reusable config contracts.

минимальный технологический стек

must have
Python 3.10+, ffmpeg/ffprobe, Whisper or Scribe, Pillow, numpy, JSON EDL, sequential render queue, QA checklist.
macOS pack
Apple `avconvert` for iPhone HDR/HLG footage, then ffmpeg renders from SDR sources with bt709 tags.
caption pack
ASS/SRT/WebVTT for portable subtitles; Pillow/numpy for PNG overlays, contact sheets and review frames.
visual editor
browser board, tracks/items state, safe-zone overlays, contact sheets and review MP4.
adapter future
OpenTimelineIO export, Remotion or HyperFrames render adapters, WebCodecs preview, MCP tools for agent control.
